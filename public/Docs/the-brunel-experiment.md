# The Brunel Experiment

## Five models. One character.

Brunel is not a model.

Brunel is one behavioral tuning of ARCHEngine.

To test what that means, we asked five different model backends the same questions.

First, each model answered as itself.

Then the same model answered through Brunel.

The result was not identical wording.

It was something more interesting.

The baseline models diverged into different voices, different instincts, and different ways of interpreting pressure.

Through ARCHEngine, they became recognizably Brunel.

Same laws.

Same moral gait.

Different temperaments.
