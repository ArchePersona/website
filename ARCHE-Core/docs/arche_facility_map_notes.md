# ARCHE Facility Map Notes

## Vision

ARCHE should be explained as a human-readable facility map, inspired by old amusement park orientation maps.

The map should feel like a place people can walk through, not a corporate architecture diagram.

## Main Areas

### Engine Room

Signal flow, agents, memory coordination, state movement, and live processing.

### Courthouse

The governance area.

The Tribunal lives inside the Courthouse.

Human-facing judge names:

- Judge Dreadlock
- Judge July
- Judge Stone

Formal roles:

- Vigilance
- Empathy
- Arbiter

### Delivery Room

The expression layer where the response is prepared for the user.

### Output Gate

The final deterministic check before anything leaves the system.

### Memory Archive

The continuity area: working memory, long-term memory, compressed fragments, and archive residue.

### Whisper Promenade

An elevated overlook where the Whisperers observe patterns across the facility.

Visual details:

- elevated walkway
- railings
- quiet figures
- old red phones
- note boards
- suggestion basket

The basket label:

SUGGESTIONS

Possible wall line:

I don’t mind if you ignore me… but I notice.

### Basement

Hidden Founder Goblin layer.

The basement entrance should be a small hatch in the lower-right corner of the map.

It should feel discovered, not advertised.

Possible labels:

- goblins only
- maintenance
- authorized salvage personnel

## Environmental Lines

## Tone

Retro. Playful. Readable. Layered. Human. Not corporate.
